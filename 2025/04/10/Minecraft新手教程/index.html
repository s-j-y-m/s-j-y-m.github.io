Minecraft新手教程

Created2025-04-10|Updated2025-04-10
|Post Views:

开始

要想流畅的安装MC JAVA版本

你必须要有以下的工具

  1. 一个正常的浏览器(推荐:EDGE)

  2. 一个正常的搜索引擎(推荐:搜索 - Microsoft 必应)

  3. 一个正常的网络环境(如果有网址无法访问,可以使用加速器或者VPN)

安装

启动器

首先,你可以选择你要使用的启动器,主流的启动器有很多

PCL

HMCL

等等一堆启动器

对于纯新手建议选择PCL,操作简单,界面明了,BUG更少(相对而言)

注意:一定要从官方的链接下载启动器,从什么3DM,下载站下的启动器不保证安全!

如果你找不到它们的官网,可以在这里跳转:
PCL: 龙腾猫跃正在创作Minecraft 启动器 PCL 和某独立游戏 | 爱发电

(PCL的下载链接在这里): PCL 最新正式版永久下载丨龙腾猫跃丨爱发电

HMCL: Hello Minecraft! Launcher

JAVA

对于JAVA的选择其实很简单,基本上可以无脑选择最新的JAVA(JDK21/JDK22)

推荐选择JDK21,基本上全版本都可以进(个别的整合包需要特定的JDK,请按照作者的要求下载)

找JDK下载实在是有点折磨了,我这里直接放链接

JDK21: Java Downloads | Oracle 中国

不懂下哪个版本的点这个: https://download.oracle.com/java/24/latest/jdk-24_windows-x64_bin.msi

可能下载的很慢,但是最好不要去第三方的网站下载,下到病毒的概率可不低!
